Octave 사용법

doc  : 전체 메뉴얼
help --list  : 모든 연산자, 키워드, ㅎ마수
help [함수명] : 함수 설명

lookfor [키워드]: 함수이름 모를때, 특정키워드가 설명에 들어간 함수를 검색
Octave --traditional : 출력형식등을 매트랩과 유사하게.
quit : 종료
exit : 종료

# 주석

#} block comment, close
